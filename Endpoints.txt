
ENDPOINTS  API

 * USERS *

GET   /user/id (int) 
input: id of user
output: json of user 

GET ALL /users
input: none
out: colection of users JSON

POST /user/create
input: ID, name, email, address, phone, id_tec, id_cli ( JSON )
out: code 201 (created)

PUT /user/update/parameter
input: parameter to be update
output: code 200 (ok)

DELETE /user/id
input: user id to remove
output: code 200(ok)
